# Șablon Proiect Student – Python + Machine Learning

Acesta este un șablon de proiect pentru studenți.

## Structură

- `data/` – seturi de date (NU puneți fișiere foarte mari în GitHub).
- `notebooks/` – notebook-uri Jupyter/Colab folosite pentru explorare și experimentare.
- `src/` – cod Python organizat pe module (logica principală a proiectului).
- `reports/` – rapoarte, grafice, rezultate finale.

## Pași recomandați pentru student

1. Creați un nou repo pe contul vostru GitHub, pornind de la acest șablon (fork sau template).
2. Deschideți notebook-ul din `notebooks/` în Google Colab.
3. Lucrați la proiect, salvați modificările și faceți **commit + push** regulat.
4. La final, asigurați-vă că:
   - codul rulează fără erori
   - notebook-ul principal este curat și bine comentat
   - există un scurt fișier de raport în `reports/`.
